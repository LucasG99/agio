# Ágio

O Ágio é uma fintech que resolve dois problemas de uma vez: a gestão de folha salarial 
para pequenas e médias empresas, que hoje é cara e burocrática, e o acesso a crédito 
justo para os colaboradores dessas empresas, que frequentemente recorrem a empréstimos 
com juros abusivos em momentos de aperto financeiro.

O modelo é B2B2C: a empresa adota o software de folha, simples e acessível, e os 
colaboradores ganham acesso a antecipação salarial e crédito consignado digital 
diretamente pelo app, com desconto automático em folha.
